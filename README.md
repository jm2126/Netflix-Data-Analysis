# Netflix-Data-Analysis

This project intends to analyse Netflix's library of TV Shows and Movies using Excel for inital data inspection/cleaning, Power BI for data visualisation and SQL for data analysis. The goal is to uncover trends and insights in its content library, including, but not limited to:

- Global distribution of movies compared to TV shows
- Content trends over time
- Top producing countries
- Growth patterns
- Most common genres and ratings
- Correlation between popular titles
